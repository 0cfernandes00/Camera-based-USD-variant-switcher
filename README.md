# Maya USD Camera-based LOD Variant Switcher

Personal Project completed Winter 2026 in Autodesk Maya wiht Python USD API 

## Demo

![Tool Demo](variant_update_new.gif)

## Introduction

This tool is intended to act similar to Unreal Engine's Nanite. It adjusts the asset's LOD based on the distance from camera.

## Setup / Usage

The logic for the tool is in variant_switcher_script.py.
